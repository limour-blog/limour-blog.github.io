## 部署
+ [反向代理服务](/Docker-bu-shu-Nginx-Proxy-Manager)
+ 注意 `OPENLIST_ADMIN_PASSWORD` 设置管理员密码
```bash
mkdir -p ~/app/olist && cd ~/app/olist

cat > ~/app/olist/docker-compose.yml <<EOF
services:
  openlist:
    image: 'openlistteam/openlist:latest'
    user: '0:0'
    environment:
      - OPENLIST_ADMIN_PASSWORD=123456
    volumes:
      - './data:/opt/openlist/data'
    restart: unless-stopped

networks:
  default:
    external: true
    name: ngpm
EOF

sudo docker compose up -d
sudo docker compose logs
```
