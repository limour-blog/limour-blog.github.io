【探索】为Next-Web开发RAG插件

Last updated on February 10, 2025 am

Dify、FastGPT、LangChain之类的用来做知识库都太过笨重，难以在1C1G的服务器上流畅运行。而 ChatGPT-Next-Web 可以在 Vercel 上免费部署，并且 NextChat-Awesome-Plugins 带来的插件功能支持 RESTful API 的调用。因此只要用 FastAPI 将嵌入、重排、Qdrant向量检索包装到一起，就可以为 Next-Web 提供一个 RAG 插件。
